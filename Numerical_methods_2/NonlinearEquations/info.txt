Метод бисекций:
    Для его применения достаточно, чтобы выполнялось f(a)*f(b) < 0
    и функция f была бы непрерывна.
    Скорость сходимости: метод сходится со скоростью геометрической
    прогрессии, знаменатель которой равен 1/2 (медленно, но надежно).
    Из оценки |x(n) - x| <= (b(n) - a(n)) / 2 = (b - a) / 2 ^ (n + 1)
    следует, что итерации стоит вести до тех пор, пока не будет выполнено
    неравенство b(n) - a(n) < 2 * epsilon. Тогда можно принять x(n) за
    приближение к корню с точностью epsilon.

Метод простой итерации:
    Достаточное условие сходимости:
    |psi'(x)| = |1 - lambda0 * f'(x)| <= gamma < 1.

Метод релаксации:
    |x(n + 1) - x(n)| = tau * |f'(x)|, где
    tau = 2 / (m + M), M = max(f'(x)), m = min(f'(x)), x из [a, b].
    Скорость сходимости: q = (M - m) / (M + m) = const;

Метод Ньютна:
    f(x) = x - f(x) / f'(x)
    Сходимость: Пусть x(k) - простой корень уравнения f(x)= 0
    (f'(x(k)) != 0), в некоторой окрестности которого функция
    y = f(x) дважды непрерывно дифференцируема.
    Тогда найдется такая малая σ -окрестность корня x(T),
    что при произвольном выборе начального приближения x(0)
    из этой окрестности итерационная последовательность метода
    Ньютона не выходит за пределы этой окрестности и справедлива
    оценка:
    x(n+1) − x(T) ≤ c * (x(n) − x(T))^2 , n ≥ 0.
    Это означает, что метод Ньютона сходится с квадратичной скоростью.
    Грубо говоря, на каждой итерации число верных знаков приближения
    примерно удваивается.
